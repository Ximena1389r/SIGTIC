Este ejemplo crea una página web simple con una estructura básica. Aquí hay una descripción de las partes principales: : Declaración del tipo de documento. : El elemento raíz que contiene todo el contenido de la página. : Contiene metadatos sobre la página, como el título que se muestra en la pestaña del navegador. : Define el título de la página que se muestra en la pestaña del navegador. <body>: Contiene el contenido visible de la página. <header>, <nav>, <main>, <footer>: Secciones estructurales de la página. <h1>, <h2>: Encabezados de diferentes niveles. <p>: Párrafos de texto. <ul>, <li>: Listas no ordenadas y elementos de lista. <a>: Enlaces a otras páginas o recursos. Este es solo un ejemplo muy básico.
